# ניתוח נתוני Titanic (EDA) עם Python

> פרויקט לימודי לניתוח אקספלורטיבי (Exploratory Data Analysis) על סט הנתונים המפורסם **Titanic** באמצעות `pandas`, `matplotlib`, ו־`seaborn`.

---

## מה יש בפרויקט?

הסקריפט `test.py`:
- טוען את הקובץ `titanic.csv`
- מדפיס סקירה ראשונית: `head()`, `info()`, נתונים חסרים, וסטטיסטיקות תיאוריות
- מפיק ויזואליזציות מרכזיות:
  - התפלגות שורדים
  - שרידות לפי מגדר
  - התפלגות גילאים
  - שרידות לפי מחלקה (`Pclass`)
  - קשר בין `Fare` לבין שרידות
  - מפת חום של קורלציות בין משתנים נומריים
- מציג מסקנות ביניים קצרות בסוף הריצה

---

## דוגמת פלט (מה תראו בהרצה)

בהרצה תקבלו:
- הדפסות לטרמינל (תצוגה ראשונית, מידע כללי, חסרים, סטטיסטיקות)
- חלונות גרפים (matplotlib) שנפתחים אחד אחרי השני

## מה אפשר לשפר/להוסיף (רעיונות להמשך)

- שמירת גרפים לקבצים (`.png`) במקום/בנוסף ל־`plt.show()`
- טיפול חכם בחסרים (למשל אימפיוטציה ל־`Age`)
- השוואת שרידות לפי:
  - משפחות (SibSp/Parch אם קיימים)
  - נמל יציאה (Embarked אם קיים)
- בניית מודל בסיסי לחיזוי שרידות (אופציונלי): Logistic Regression / Random Forest

## תקלות נפוצות

### `FileNotFoundError: [Errno 2] No such file or directory: 'titanic.csv'`

- ודאו שהקובץ `titanic.csv` נמצא באותה תיקייה של `test.py`
- ודאו שאתם מריצים מה־Working Directory הנכון

### הגרפים לא נפתחים / נתקעים

- נסו להריץ מתוך טרמינל (ולא רק בתוך סביבת Notebook)
- ב־IDE: ודאו שמצב ה־Python שלכם תומך בפתיחת חלונות גרפים
